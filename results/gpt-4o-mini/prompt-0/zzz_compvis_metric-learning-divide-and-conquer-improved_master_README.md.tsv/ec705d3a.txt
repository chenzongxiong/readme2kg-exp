# Improving Deep Metric Learning by Divide and Conquer 
## About  
PyTorch implementation for the paper _Improving Deep Metric Learning by Divide and Conquer_ accepted to **<PUBLICATION>TPAMI</PUBLICATION>** (Sep. 2021), which is our follow-up paper of [_Divide and Conquer the Embedding Space for Metric Learning (<CONFERENCE>CVPR 2019</CONFERENCE>)](https://github.com/CompVis/metric-learning-divide-and-conquer)  
**Links**: * arxiv: https://arxiv.org/abs/2109.04003 or * <PUBLICATION>TPAMI</PUBLICATION> early access: https://ieeexplore.ieee.org/document/9540303  
## Requirements  
* <SOFTWARE>PyTorch</SOFTWARE> 1.1.0  
* <SOFTWARE>Faiss-GPU</SOFTWARE> >= 1.5.0, [Link](https://github.com/facebookresearch/faiss)  
* <SOFTWARE>albumentations</SOFTWARE> >= 0.4.5, [Link](https://github.com/albumentations-team/albumentations)  
## Usage  
### Training:  
Training is done by calling `python train.py` and setting the respective params, all of which are listed and explained in `/experiment/margin_loss_resnet50.py` (the default setup for all our experiments).
